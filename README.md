# Métodos de Arrays com JavaScript
Acesse >> https://vitormendonca.github.io/metodos-array-javascript/

Projeto interativo feito com HTML, CSS e JavaScript puro para praticar os principais métodos de arrays e manipulação do DOM.

A página apresenta uma lista de alunos e exercícios sobre métodos como `map()`, `filter()`, `find()`, `forEach()`, `reduce()`, `some()` e `every()`. Ao clicar no botão de cada pergunta, a resposta aparece em formato de bloco de código.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Conceitos praticados

- Array de objetos
- Métodos de array
- Manipulação do DOM
- `querySelectorAll()`
- `createElement()`
- `appendChild()`
- `addEventListener()`
- Dropdown com JavaScript
- Exibição dinâmica de respostas

## Exercícios incluídos

- Criar uma lista com os nomes dos alunos usando `map()`
- Filtrar alunos ativos usando `filter()`
- Encontrar o primeiro aluno com nota abaixo de 70 usando `find()`
- Exibir frases no console com `forEach()`
- Somar mensalidades usando `reduce()`
- Verificar se existe aluno inativo usando `some()`
- Verificar se todos os alunos estão ativos usando `every()`
