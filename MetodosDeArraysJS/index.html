Métodos de arrays com JS

Vamos praticar os métodos de arrays em JavaScript usando a lista de alunos ao lado.

  • .map

    Crie uma nova lista apenas com os nomes dos alunos.

  • .filter

    Crie uma nova lista contendo apenas os alunos ativos.

  • .find

    Encontre o primeiro aluno com nota abaixo de 70.

  • .forEach

    Mostre no console uma frase para cada aluno:

    João concluiu 12 aulas.

    Maria concluiu 18 aulas.

    Karim concluiu 25 aulas.

    Pedro concluiu 7 aulas.

    Ana concluiu 20 aulas.

  • .reduce

    Some o total de mensalidades.

  • .some

    Verifique se existe algum aluno inativo.

  • .every

    Verifique se todos os alunos estão ativos.
